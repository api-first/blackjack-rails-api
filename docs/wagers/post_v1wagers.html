Wagers API

POST /v1/wagers

POST /v1/wagers

Parameters

Name Description
type Must always be set to <code>wagers</code>.
relationships[hand] The hand that the wager is being placed on.
relationships[player] The player that is placing the wager.
attributes[kind] The kind of wager that is being placed. Options are `intitial`.
attributes[amount] The amount of the wager, as a decimal.

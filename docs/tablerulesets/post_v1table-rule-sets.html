TableRuleSets API

POST /v1/table-rule-sets

POST /v1/table-rule-sets

Parameters

Name Description
type Must always be set to <code>table-rule-sets</code>.
attributes[name] The name of the rule set. This may be used to describe the type of game being offered at a table.
attributes[deck-count] The number of decks that will be used at the table.
attributes[player-position-count] The number of playing positions at the table. Must be between 5 and 7.
attributes[maximum-players-per-position] The maximum number of players that can play from an individual position on the table.
attributes[maximum-bets-per-player] The maximum number of bets that a player can make at a table. This is naturally limited by the number of positions, but could be lower.
attributes[no-hole-card] If <code>true</code>, the dealer does not deal themselves a hole card during the initial deal. Defaults to <code>false</code>.
attributes[original-bets-only] This boolean can only be set to true or false if <code>no-hole-card</code> is <code>true</code>. When <code>original-bets-only</code> is true, if the player's hand loses to dealer blackjack, only the mandatory initial bet is forfeited, and all optional bets (doubles and splits) are pushed.
attributes[initial-player-card-face-up] This boolean indicates whether the initial card dealt to players is face up. It defaults to <code>true</code>.
attributes[blackjack-pay-out] The amount that a winning blackjack hand pays out as a multiple of the original bet.
attributes[standard-pay-out] The amount that a standard winning hand pays out as a multiple of the original bet.
attributes[dealer-stands-at] The total at which the dealer must stand. Defaults to 17.
attributes[dealer-draws-if-soft-stand-at] A boolean that if <code>true</code> requires the dealer to hit on the <code>dealer-stands-at</code> total if the total is soft.
attributes[surrender] An enum that indicates whether surrender is offered and if so, how. Valid values include <code>not_allowed</code>, <code>early</code>, and <code>late</code>. Defaults to <code>not_allowed</code>".
attributes[surrender-percentage] A decimal that indicates the percentage of the original bet that is surrendered when the player makes that choice. Must be <code>NULL</code> if <code>surrender</code> is <code>not_allowed</code>
attributes[insurance-available] A boolean that indicates whether insurance side bets are offered. Defaults to <code>true</code>.
attributes[insurance-pay-out] A decimal that is the multiple of the original bet that is paid if an insurance side bet is won. Must be <code>NULL</code> if <code>insurance-available</code> is <code>false</code>.
attributes[dealer-match-available] A boolean that indicates whether dealer match side bets are offered. Defaults to <code>true</code>.
attributes[dealer-match-pay-out] A decimal that is the multiple of the original bet that is paid if a dealer match side bet is won. Must be <code>NULL</code> if <code>dealer-match-available</code> is <code>false</code>.
attributes[double-down-minimum-percentage] A decimal that is the multiple of the original bet that is the minimum bet in "double down" situations. Defaults to <code>1.0</code>.
attributes[double-down-maximum-percentage] A decimal that is the multiple of the original bet that is the maximum bet in "double down" situations. Defaults to <code>1.0</code>.
attributes[split-identical-ranks-only] A boolean that indicates if a player can only split identical ranks (for example, 2 Jacks) or if they can split as long as the value of the rank is the same. Defaults to <code>false</code>
attributes[maximum-double-count-per-hand] An integer that indicates the maximum number of times that a player can double down per original hand. Defaults to NULL (no maximum).
attributes[maximum-split-count-per-hand] An integer that indicates the maximum number of times that a player can split per original hand. Defaults to NULL (no maximum).
attributes[may-hit-split-aces] A boolean that indicates if the player may hit, split, or double split aces. Defaults to <code>false</code>.
attributes[non-controlling-player-chooses-split-hand-to-bet-if-not-following] A boolean that indicates whether a player that does not choose to follow a split by the controlling player is allowed to choose which of the split hands to bet. Defaults to <code>false</code> by default.
attributes[player-must-stand-on-soft-21] A boolean that indicates if a player must stand on a soft 21. Defaults to <code>true</code>.
attributes[restrict-doubling-to-hard-totals] An array that includes the totals that can be doubled. Defaults to <code>NULL</code> (which means there are no restrictions).
attributes[dealer-wins-ties] A boolean that indicates if a dealer wins tied hands. Otherwise the hands push (nobody wins). Defaults to <code>false</code>.
attributes[round-initial-betting-window-seconds] An integer that indicates the number of seconds that the initial betting window will last for each round at the table.
attributes[minimum-wager-amount] An integer that indicates the minimum initial wager amount.
